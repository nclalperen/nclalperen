## Featured projects

| Project | What it is |
| --- | --- |
| [**emwave-c**](https://github.com/nclalperen/emwave-c) | Real-time 2D FDTD electromagnetic wave simulator in C11 — OpenMP-parallel solver, CPML boundaries, SDL2 + Dear ImGui front-ends, headless S-parameter sweeps |
| [**projecttabletop**](https://github.com/nclalperen/projecttabletop) | Okey 101 (Turkish tile game) in Godot 4.6 — deterministic action/validator/reducer rules core, tiered bot AI, EOS online multiplayer, 75+ headless tests |
| [**game-tracker**](https://github.com/nclalperen/game-tracker) | Open-source desktop + web game library tracker — React/Vite web app in a Tauri (Rust) shell, offline-first, metadata enrichment, optional local LLM suggestions |
| [**fpga-rtc**](https://github.com/nclalperen/fpga-rtc) | VHDL real-time clock on an Artix-7 (Nexys A7-100T) — clock dividers, 7-segment multiplexing, Vivado batch flow, Icarus Verilog CI |
| [**plc-thesis**](https://github.com/nclalperen/plc-thesis) | Bachelor thesis: measuring how EMI, distance, and heat degrade powerline (HomePlug-AV) networking, with oscilloscope captures and a MATLAB analysis pipeline |
